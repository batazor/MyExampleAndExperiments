# fluentd-compose

Example use fluentd, elasticsearch, docker on docker-compose

### getting started

```
docker-compose build
docker-compose up
```

### Services

| Name              | URL                     |
|-------------------|-------------------------|
| Nginx             | localhost:9999          |
| Node Example      | localhost:4040          |
| MongoDB           | localhost:27017         |
| Kibana            | localhost:5601          |
